Escape the vault :
Escape the Vault is a simple Python guessing game where the player must guess a secret vault code before running out of attempts. The project demonstrates fundamental programming concepts such as functions, loops, conditional statements, input validation, and random number generation.

Each time the game starts, a secret vault code is randomly generated. The player must guess the correct number within a limited number of attempts, depending on the selected difficulty level. After each guess, the game provides hints indicating whether the guess was too high or too low.

The project also includes a flowchart created before writing the final code, showing the game logic and decision structure

System Requirements:
•	Any operating system that supports Python (Windows, macOS, or Linux)

Software Requirements:
•	Python 3.x
•	A code editor or environment such as:
o	Jupyter Notebook
o	VS Code
o	PyCharm

Libraries Used:
This project only uses one built-in Python library:
•	random – used to generate the secret vault code.
No external libraries are required.

How to play
1.	Run the game.
2.	Choose a difficulty level (easy, medium, or hard).
3.	Enter guesses when prompted.
4.	The game will tell you if your guess is too high or too low.
5.	If you guess the correct number, the vault opens and you escape.
6.	If you run out of attempts, the alarm triggers and the vault remains locked.
At the end of each round, the player can choose whether to play again or exit the game.
